## Our Dual-Engine Architecture

The KnDo Network runs two consensus engines in parallel over a unified state layer to give you the best of both worlds:

    * **Nexa UTXO Engine (PoW):** Handles native token issuance, ultra-fast payments, and hardware-accelerated transaction validation using Schnorr signatures with 1-second block times.
    * **Avalanche Subnet Engine (EVM + PoS):** A fully EVM-compatible execution environment powered by Avalanche's Snow consensus. It delivers probabilistic finality in under 2 seconds alongside full Solidity smart contract support.
